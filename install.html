  • What software is needed? How would I run the website locally?

    Use the following npm commands in order to set up the environment locally.

    • npm install tailwindcss @tailwindcss/vite
    • npm install lucide-react react-router-dom tailwind-merge
    • npm install @radix-ui/react-toast class-variance-authority clsx
    • npm install @auth0/auth0-react@1.12.1
    • npm install @google/generative-ai
    • npm install express cors dotenv @google/generative-ai
    • npm install --save-dev concurrently
    • npm install --save-dev @types/express @types/cors
    • npm install --save-dev gh-pages
    • npm install jspdf

    Put the .env files given to Dr. Rahimi and Dr. Atkison inside the project on the root level, and you will be able to develop locally

    In order to run the website locally, use npm run dev

    In order to deploy the website to the url above, use npm run build followed by npm run deploy
